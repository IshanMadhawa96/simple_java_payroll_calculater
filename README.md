# simple_java_payroll_calculater
<br>
How to Configuer Projec
<br>
1.Clone Project From Git
<br>
2.Create Database and Run SQL query(Here i used MS SQL Server)
<br>
3.Create ODBc Data Sourse in Windows Control Panel
<br>
4.Install MS SQL connector jar to Java Project
<br>
5.Run and Enjoy Project
